# Merging Companies Frontend
Threw this together to make it a little easier to interact with the merging system.

## Installation & Running
Once you have cloned this repo to your local machine, run `npm install`, and then `npm run dev`. This will get the front-end up and running and will allow you to interact with merging of companies.

The dev server is set to run on port `5173`, and will always launch on that port unless changed within the `vite.config.ts`.
